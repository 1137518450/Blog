# 1、变量和常量

变量： 存储信息的，日后被调用、修改操作
常量： 固定不变的量，字母大写

# 2、变量的命名规则

（1）. 字母数字下划线组成
（2）. 不能以数字开头，不能含有特殊字符和空格
（3）. 不能以保留字命名
（4）. 不能以中文命名 
（5）. 定义的变量名应该有意义
（6）. 驼峰式命、 下划线分割单词
（7）. 变量名区分大小写
